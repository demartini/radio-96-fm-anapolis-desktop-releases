## Sobre o aplicativo

A **Rádio 96 FM Anápolis** é a FM Oficial de Goiás — referência em entretenimento, notícia e música para toda a região.
Este aplicativo foi criado para quem escuta a rádio todos os dias e quer uma forma rápida, estável e sem distrações de ouvir a programação diretamente no computador.

<p align="right">(<a href="#top">voltar ao topo</a>)</p>

## Recursos

Tudo o que você precisa para ouvir rádio no desktop, sem complicações:

- 🎙️ **Streaming ao vivo** — conexão direta com a transmissão com reconexão automática em caso de queda.
- 🎚️ **Controle de volume** — ajuste e silenciamento com persistência entre sessões.
- 🌙 **Tema claro e escuro** — segue o tema do sistema ou configurável manualmente.
- 🖥️ **Bandeja do sistema** — controle rápido sem precisar abrir a janela.
- ⌨️ **Atalhos globais** — `Ctrl/Cmd + Shift + P/M/↑/↓` mesmo com o app em segundo plano.
- 🚀 **Iniciar com o sistema** — a rádio já está tocando quando você liga o computador.
- 🔔 **Atualizações automáticas** — o app se mantém sempre atualizado.
- 🪶 **Leve e rápido** — interface nativa com consumo mínimo de memória.

## Download

Downloads oficiais e seguros — todos os instaladores são assinados digitalmente.
Compatível com macOS, Windows e Linux.

### macOS

| Arquivo                                               | Arquitetura                       | Requisito                        |
| ----------------------------------------------------- | --------------------------------- | -------------------------------- |
| [Radio.96.FM.Anapolis_1.0.0_universal.dmg ↓][dmg-url] | Universal (Apple Silicon + Intel) | macOS 10.15 Catalina ou superior |

**Como instalar:**

Abra o `.dmg`, arraste o app para a pasta **Aplicativos** e execute.
Na primeira abertura, clique com o botão direito no ícone e escolha **Abrir** para permitir a execução recomendada pelo macOS.

### Windows

| Arquivo                                               | Tipo                            | Arquitetura |
| ----------------------------------------------------- | ------------------------------- | ----------- |
| [Radio.96.FM.Anapolis_1.0.0_x64-setup.exe ↓][exe-url] | Instalador padrão (recomendado) | x64         |
| [Radio.96.FM.Anapolis_1.0.0_x64_en-US.msi ↓][msi-url] | Instalador corporativo (MSI)    | x64         |

Execute o instalador e siga as instruções na tela.

### Linux

| Arquivo                                                     | Formato       | Indicado para                  |
| ----------------------------------------------------------- | ------------- | ------------------------------ |
| [Radio.96.FM.Anapolis_1.0.0_amd64.AppImage ↓][appimage-url] | AppImage      | Qualquer distro (recomendado)  |
| [Radio.96.FM.Anapolis_1.0.0_amd64.deb ↓][deb-url]           | Debian/Ubuntu | Ubuntu, Debian, Mint, Pop!_OS  |
| [Radio.96.FM.Anapolis-1.0.0-1.x86_64.rpm ↓][rpm-url]        | Fedora/RHEL   | Fedora, openSUSE, RHEL, CentOS |
